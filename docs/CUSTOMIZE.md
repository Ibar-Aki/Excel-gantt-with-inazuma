# カスタマイズ

InazumaGantt.basの定数を変更して調整できます。

## 期間と行数
- GANTT_DAYS: ガントチャートの日数
- DATA_ROWS_DEFAULT: データ入力の初期行数

## シート名/凡例
- MAIN_SHEET_NAME: メインシート名
- HOLIDAY_SHEET_NAME: 祝日マスタ名
- GUIDE_SHEET_NAME: 説明シート名
- GUIDE_LEGEND_START_CELL: 凡例の開始セル

## 色設定
- COLOR_PLAN / COLOR_PROGRESS / COLOR_ACTUAL / COLOR_TODAY
- COLOR_HOLIDAY / COLOR_ROW_BAND / COLOR_WARN / COLOR_ERROR
- COLOR_INAZUMA: イナズマ線の色

変更後はSetupInazumaGanttを再実行してください。
