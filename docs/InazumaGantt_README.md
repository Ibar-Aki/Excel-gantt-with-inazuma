# イナズマガントチャート 取扱説明書

## 概要
「イナズマ線」で進捗の遅れ・前倒しを把握します。

- 予定通り: イナズマ線が今日の赤線と一致
- 進んでいる: イナズマ線が右側
- 遅れている: イナズマ線が左側

---

## 導入手順
1. ExcelでAlt+F11を押してVBAエディタを開く
2. プロジェクトを右クリック → [挿入] → [標準モジュール]
3. InazumaGantt.basの内容を貼り付け
4. エディタを閉じてシートへ戻る

---

## 使い方
### 1) 初期セットアップ
SetupInazumaGanttを実行すると以下が自動設定されます。
- タイトル/ヘッダー
- 日付列（120日分）
- 入力規則（進捗率・状況）
- 罫線
- 祝日マスタと説明シート（凡例付き）

### 2) データ入力
A-K列に以下を入力します。

| 列 | 内容 |
|---|---|
| A | No. |
| B | 階層 |
| C | タスク |
| D | タスク詳細 |
| E | 状況（未着手 / 進行中 / 完了 / 保留） |
| F | 進捗率（0.7 または 70%） |
| G | 担当 |
| H | 開始予定 |
| I | 完了予定 |
| J | 開始実績 |
| K | 完了実績 |

### 3) 描画更新
RefreshInazumaGanttを実行すると以下が更新されます。
- 予定バー（灰）/進捗バー（青）
- 実績バー（緑の太線）
- イナズマ線（オレンジ）
- 今日線（赤）

---

## カレンダー/ヘッダーの位置
- 週ヘッダー: 6行目
- 曜日表示: 7行目（L列以降）
- 日付行: 8行目（L列以降）
- データ開始: 9行目

---

## 見方
- 赤い縦線: 今日
- オレンジの折れ線: 各タスクの進捗位置
  - 赤線より右: 予定より進んでいる
  - 赤線より左: 予定より遅れている

---

## シート構成
- InazumaGantt : メインシート
- 祝日マスタ : 祝日一覧（A列に日付）
- InazumaGantt_説明 : 使い方と凡例

---

## マクロ一覧
| マクロ名 | 機能 |
|---|---|
| SetupInazumaGantt | 初期設定（ヘッダー、日付、入力規則、罫線） |
| DrawGanttBars | 予定/進捗バーの描画（条件付き書式） |
| DrawActualBars | 実績バーの描画（緑の太線） |
| DrawInazumaLine | イナズマ線と今日線の描画 |
| RefreshInazumaGantt | 上記をまとめて更新 |
