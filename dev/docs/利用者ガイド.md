# イナズマガントチャート 利用ガイド

## はじめに

このガイドでは、イナズマガントチャートの使い方を説明します。
マクロの設定はすでに完了しており、すぐにお使いいただけます。

---

## 画面構成

| エリア | 説明 |
|---|---|
| **タイトル (1行目)** | 「イナズマガントチャート」と表示 |
| **ボタン (2行目)** | 3つの操作ボタン |
| **情報エリア (K-L列)** | 開始日・週表示・今日の日付 |
| **ヘッダー (8行目)** | 列名（LV, No., TASK など） |
| **データ行 (9行目〜)** | タスクを入力するエリア |
| **ガントチャート (O列〜)** | 進捗バー・イナズマ線を表示 |

---

## 操作ボタン

| ボタン | 機能 |
|---|---|
| **ガント更新** | ガントチャートを最新の状態に更新します |
| **土日切替** | 土日列の表示/非表示を切替えます |
| **書式リセット** | 罫線や書式が崩れた時に修復します |

---

## タスクの入力方法

### 基本入力

1. **No.** - 自動で1〜400が入っています
2. **LV** - 階層レベル（自動判定されます）
3. **TASK (C-F列)** - タスク名を入力
   - C列に入力 → LV1（親タスク）
   - D列に入力 → LV2（子タスク）
   - E列に入力 → LV3
   - F列に入力 → LV4
4. **タスク詳細 (G列)** - 補足情報
5. **状況 (H列)** - 未着手/進行中/完了
6. **進捗率 (I列)** - 0%〜100%
7. **担当 (J列)** - 担当者名
8. **開始予定 (K列)** - 開始予定日
9. **完了予定 (L列)** - 完了予定日
10. **開始実績 (M列)** - 実際の開始日
11. **完了実績 (N列)** - 実際の完了日

---

## ダブルクリックで完了

タスクを完了させるには、**No.列 または LV列をダブルクリック**します。

- 状況が「完了」に変わります
- 進捗率が100%になります
- 完了実績に今日の日付が入ります

> **注意**: すでに完了しているタスクはダブルクリックしても変更されません

## ガントチャートの見方

### バーの種類

| バー | 色 | 説明 |
|---|---|---|
| **予定バー** | 薄い灰色 | 開始予定〜完了予定の期間 |
| **進捗バー** | 紺色 | 進捗率に応じた長さ |
| **実績バー** | 緑色 | 実際の作業期間 |

### 線の種類

| 線 | 色 | 説明 |
|---|---|---|
| **今日線** | 赤色 | 今日の日付を示す縦線 |
| **イナズマ線** | オレンジ | 各タスクの進捗を結んだ折れ線 |

---

## イナズマ線について

イナズマ線は、プロジェクト全体の進捗を視覚化する線です。

- **今日線より左** → 進捗が遅れている
- **今日線と同じ位置** → 予定通り
- **今日線より右** → 進捗が進んでいる

## よくある質問

### Q. ガントチャートが更新されない

**A.** 「ガント更新」ボタンをクリックしてください。

### Q. 土日を非表示にしたい

**A.** 「土日切替」ボタンをクリックしてください。

### Q. 罫線が崩れた

**A.** 「書式リセット」ボタンをクリックしてください。

### Q. 完了にしたタスクを戻したい

**A.** 状況列を直接編集して「進行中」などに変更してください。
